Git

- Pra que serve: metodologia de versionamento de código.

- Como clonar um repositório: No GitHub, clicar em Code > Clone with HTTPS. Depois ir em CMD, 
localizar a pasta desejada e digitar o comando "git clone 'colar endereço HTTPS'".

- Como fazer staging das alterações: No CMD usar o comando "git init"; para adicinar os arquivos 
do diretório para o git usar o comando "git add."; usar o comando "git commit -m" + uma mensagem
de identificação desse commit; depois de alterar o arquivo é necessário repetir os comandos 
"git add." e o "git commit -m" para salvar uma nova versão (Staging).

- Como fazer um commit: usar o comando "git commit -m" + uma mensagem de identificação desse commit.

- Como fazer um push: usar o comando "git push" que vai salvar os arquivos do Git para o GitHub.

- Como fazer um pull: usar o comando "git pull" que serve para baixar o diretório atualizado do GitHub.
